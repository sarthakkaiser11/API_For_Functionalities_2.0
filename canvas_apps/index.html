Data Observability Hub

Monitor freshness, quality, and statistical profiles of your Databricks tables.

⏱️

Freshness

Track data latency, versioning, and modification timestamps. Know exactly how fresh your data is.

🛡️

Quality

Deep-dive into column-level health metrics — null percentages, distinct counts, and data fingerprints.
📊

Profile

Explore statistical distributions — mean, median, quantiles, and row counts for numeric features.

📄

Description

AI-generated documentation — table overviews, column definitions, and business context.

📌 Usage

  • Freshness → /stats/freshness/{catalog}.{schema}.{table}
  • Quality → /stats/quality/{catalog}.{schema}.{table}
  • Profile → /stats/profile/{catalog}.{schema}.{table}
  • Description → /stats/description/{catalog}.{schema}.{table}
